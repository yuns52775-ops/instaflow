Meta 비즈니스 계정 확인
광고를 집행 중이라면 이미 준비된 단계입니다
1

비즈니스 계정 여부 확인

Facebook 광고 관리자에서 광고를 집행 중이라면 비즈니스 계정이 이미 있는 것입니다. 별도 생성이 필요 없습니다.

business.facebook.com
2

처음 광고를 시작하는 경우

Meta Business Suite에 접속 후 새 비즈니스 포트폴리오 만들기를 클릭합니다. 이름과 이메일만 입력하면 바로 생성됩니다.

이미 Facebook 페이지나 인스타그램 광고를 운영 중이라면 이 단계는 건너뛰셔도 됩니다.
2
광고 계정 확인
AdPulse 연결 시 사용할 광고 계정을 확인합니다
1

광고 관리자 접속

아래 링크에서 현재 운영 중인 광고 계정을 확인합니다.

facebook.com/adsmanager
2

광고 계정 선택 확인

화면 좌측 상단 드롭다운에서 연결할 광고 계정을 확인합니다. 여러 계정이 있다면 원하는 계정을 선택해 두세요.

광고 계정 ID 위치광고 관리자 좌측 상단 또는 설정 → 광고 계정 → 계정 ID
3

계정 권한 확인

본인이 만든 계정이거나, 비즈니스 관리자로부터 관리자 또는 광고주 역할을 부여받은 계정이어야 합니다.

결제 수단 등록 필요: 광고 계정에 결제 수단이 등록되어 있어야 API로 데이터를 불러올 수 있습니다.
3
AdPulse 로그인 및 연결
Meta 계정으로 로그인하고 광고 계정을 선택합니다
1

AdPulse 접속

AdPulse 사이트에 접속합니다. 처음에는 샘플 데이터가 표시됩니다.

AdPulse 열기
2

Meta Ads 연결 버튼 클릭

우측 상단 또는 사이드바 하단의 Meta Ads 연결 버튼을 클릭합니다.

Meta로 로그인 팝업 표시Facebook 계정으로 로그인하고 요청하는 권한을 허용합니다
3

광고 계정 선택

로그인 후 내 광고 계정 목록이 표시됩니다. 데이터를 볼 계정을 선택합니다.

4

연결 완료

연결 후 대시보드, 캠페인, 광고세트, 인사이트 등 모든 페이지에 실시간 데이터가 표시됩니다.

연결 상태는 자동 저장됩니다. 브라우저를 닫았다가 다시 열어도 재로그인 없이 바로 사용할 수 있습니다.
4
Meta 픽셀 설정
전환(구매, 회원가입 등)을 추적하려면 픽셀을 웹사이트에 설치합니다
1

이벤트 관리자 접속

Meta 이벤트 관리자에서 픽셀을 생성합니다.

business.facebook.com/events_manager
2

픽셀 생성

데이터 소스 연결 → 웹 → Meta 픽셀 선택 후 픽셀 이름을 입력하고 생성합니다.

3

픽셀 설치

생성된 픽셀 코드를 웹사이트의 모든 페이지 <head> 안에 붙여넣습니다. Shopify, WordPress, GTM을 사용 중이라면 플러그인으로 더 쉽게 설치할 수 있습니다.

플랫폼별 간편 설치 지원Shopify · WordPress · Wix · Cafe24 · GTM 등 파트너 연동 선택 가능
4

전환 이벤트 추가

구매, 장바구니 담기, 회원가입 등 주요 행동에 이벤트 코드를 추가합니다.

이벤트언제 발생주요 업종
Purchase구매 완료쇼핑몰 필수
AddToCart장바구니 추가쇼핑몰
Lead문의 / 신청 완료B2B, 교육, 금융
CompleteRegistration회원가입 완료앱, 서비스
ViewContent상품 페이지 조회공통
InitiateCheckout결제 페이지 진입쇼핑몰
Chrome 확장 프로그램 Meta Pixel Helper를 설치하면 픽셀이 정상 작동하는지 바로 확인할 수 있습니다.
자주 묻는 질문
연결 전후 자주 발생하는 궁금증
데이터가 전혀 안 나와요
광고 계정에 집행 이력이 없거나 계정이 일시정지 상태일 경우 데이터가 없을 수 있습니다. 광고 관리자에서 계정 상태와 캠페인 집행 여부를 먼저 확인해 주세요.
연결했는데 캠페인이 일부만 보여요
최근 30일 기준으로 활동이 있는 캠페인을 기본으로 표시합니다. 날짜 범위를 변경하거나 새로고침 버튼을 눌러보세요.
여러 광고 계정을 동시에 볼 수 있나요?
현재는 한 번에 하나의 광고 계정을 조회합니다. 다른 계정을 보려면 상단 연결 버튼으로 계정을 전환할 수 있습니다.
픽셀 페이지에 이벤트가 안 보여요
픽셀이 해당 광고 계정에 연결되어 있어야 합니다. Meta 이벤트 관리자 → 픽셀 → 설정 → 광고 계정 연결 항목을 확인하세요.
로그인했는데 권한 오류가 나요
로그인 팝업에서 요청하는 모든 권한에 허용을 눌러야 합니다. 일부 권한을 거부한 경우 재연결 버튼을 눌러 다시 로그인해 주세요.
다른 사람 광고 계정도 볼 수 있나요?
내 계정에 접근 권한이 있는 광고 계정만 조회할 수 있습니다. 대행사의 경우 광고 관리자에서 계정 접근 권한을 받으면 해당 계정도 선택할 수 있습니다.
