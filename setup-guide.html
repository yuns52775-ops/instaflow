Meta 광고, 처음부터
차근차근 시작하기

Facebook 계정 하나도 없는 상태에서 시작합니다.
순서대로 따라오시면 AdPulse로 실시간 광고 데이터를 볼 수 있습니다.

총 8단계 처음 설정 약 30~40분 PC 환경 권장
1
Facebook 계정 만들기
Meta 광고의 모든 것은 Facebook 계정에서 시작합니다
1

Facebook 회원가입 페이지 접속

아래 버튼을 눌러 Facebook 회원가입 페이지로 이동합니다.

facebook.com 회원가입
2

정보 입력

이름, 휴대폰 번호 또는 이메일, 비밀번호, 생년월일을 입력하고 가입하기를 클릭합니다.

이름실명 입력 (광고 계정 인증에 사용됨)
휴대폰 / 이메일자주 사용하는 연락처 입력
비밀번호8자 이상, 영문+숫자 조합 권장
3

이메일 / 전화 인증

가입 후 발송된 인증 코드를 입력하여 계정을 활성화합니다.

4

프로필 사진 설정 (선택)

광고 계정 신뢰도를 높이기 위해 프로필 사진을 설정해 두는 것이 좋습니다.

이미 Facebook 계정이 있다면 이 단계는 건너뛰고 Step 2로 이동하세요.
2
Meta Business Suite 설정
광고 계정과 자산을 관리하는 비즈니스 공간을 만듭니다
1

Meta Business Suite 접속

Step 1에서 만든 Facebook 계정으로 로그인한 상태에서 접속합니다.

business.facebook.com
2

새 비즈니스 포트폴리오 만들기

처음 접속하면 "새 비즈니스 포트폴리오 만들기" 버튼이 나타납니다. 클릭합니다.

비즈니스 이름회사명 또는 브랜드명
내 이름담당자 실명
비즈니스 이메일업무용 이메일 주소
3

비즈니스 세부 정보 입력

국가 대한민국, 주소, 웹사이트 URL을 입력합니다. 웹사이트가 없으면 인스타그램 주소를 입력해도 됩니다.

4

생성 완료 확인

비즈니스 포트폴리오가 만들어지면 Business Suite 홈 화면으로 이동됩니다.

성공 화면좌측 상단에 비즈니스 이름이 표시되면 완료
같은 이메일로 여러 포트폴리오 생성 불가. Facebook 계정 1개당 비즈니스 포트폴리오 최대 2개까지 생성됩니다.
3
Facebook 비즈니스 페이지 만들기
광고를 집행하려면 비즈니스 페이지가 반드시 필요합니다
1

페이지 만들기 이동

Facebook에 로그인 후 좌측 메뉴 → 페이지 → 새 페이지 만들기를 클릭합니다. 또는 아래 링크로 바로 이동합니다.

페이지 만들기
2

페이지 기본 정보 입력

페이지 이름브랜드명 또는 상호명
카테고리업종에 맞는 카테고리 선택 (예: 의류, 식음료)
설명브랜드/제품 소개 한 줄 (선택)
3

프로필 사진 및 커버 사진 업로드

로고 이미지를 프로필 사진으로 사용하세요. 광고 노출 시 페이지 사진이 함께 표시됩니다.

프로필 사진170×170px 이상, PNG/JPG
커버 사진820×312px 권장
4

Business Suite에 페이지 연결

Business Suite → 설정 → 계정 → 페이지 → 추가 → 방금 만든 페이지 선택

Business Suite 페이지 설정
이미 운영 중인 페이지가 있다면 새로 만들 필요 없이 기존 페이지를 Business Suite에 추가하면 됩니다.
4
광고 계정 만들기
실제 광고를 집행하고 데이터를 조회하는 계정입니다
1

광고 계정 설정 페이지 이동

Business Suite 설정 → 계정 → 광고 계정으로 이동합니다.

광고 계정 설정
2

새 광고 계정 추가

추가 → 새 광고 계정 만들기를 클릭합니다.

광고 계정 이름브랜드명_광고 (예: 나이키_광고)
시간대아시아/서울 (UTC+9)
통화KRW (한국 원)
3

광고 계정 ID 확인 및 메모

생성된 광고 계정을 클릭하면 광고 계정 ID가 표시됩니다. 나중에 AdPulse 연결에 필요하므로 메모해 두세요.

📋 광고 계정 ID 예시: act_1234567890123456
숫자 부분만 메모해도 됩니다. AdPulse가 자동으로 act_를 붙여줍니다.

4

광고 계정에 사람 추가 (팀이 있을 경우)

광고 계정 → 사람 추가 → 이메일로 팀원 초대 → 역할 설정 (관리자 / 광고주 / 분석가)

시간대와 통화는 한 번 설정하면 변경이 어렵습니다. 반드시 처음에 정확하게 설정하세요. 한국에서 운영한다면 KRW + 서울 시간대를 선택하세요.
5
결제 수단 등록
결제 수단이 없으면 광고 집행도, 데이터 조회도 되지 않습니다
1

결제 설정 페이지 이동

광고 관리자 → 좌측 메뉴 결제 → 또는 아래 링크로 바로 이동합니다.

결제 수단 설정
2

결제 수단 추가

결제 수단 추가 클릭 후 사용할 방식을 선택합니다.

신용카드 / 체크카드Visa, Mastercard, 국내 카드 대부분 지원. 가장 보편적인 방법
PayPal해외 결제용. 국내에서는 카드를 더 많이 사용
3

결제 한도 설정

계정 지출 한도를 설정하면 예상치 못한 과금을 방지할 수 있습니다. 결제 → 계정 지출 한도 설정에서 월별 최대 금액을 입력하세요.

해외 결제가 막혀 있다면? 카드 발급사에 해외 온라인 결제 허용을 요청하거나, 해외 결제가 기본 허용된 카드(신한·하나·우리 VISA 등)를 사용하세요.
6
인스타그램 계정 연결
인스타그램 광고를 집행하려면 계정을 비즈니스 자산에 연결해야 합니다
1

인스타그램 계정을 비즈니스 계정으로 전환

인스타그램 앱 → 프로필 → 설정 → 계정 → 비즈니스 계정으로 전환

개인 계정으로는 광고 집행이 불가합니다. 반드시 비즈니스 계정으로 전환해야 합니다.
2

Business Suite에 인스타그램 연결

Business Suite → 설정 → 계정 → Instagram 계정 → 추가 클릭

Instagram 계정 설정
3

인스타그램 로그인으로 연결 승인

인스타그램 아이디와 비밀번호를 입력하여 연결을 완료합니다. 연결되면 인스타그램 피드/스토리 광고를 집행할 수 있습니다.

인스타그램 광고를 집행하지 않고 Facebook 광고만 집행할 경우 이 단계는 건너뛸 수 있습니다.
7
AdPulse 로그인 및 광고 계정 연결
이제 AdPulse에 Meta 계정을 연결해 실시간 데이터를 봅니다
1

AdPulse 접속

아래 버튼으로 AdPulse를 엽니다. 처음에는 샘플 데이터가 표시됩니다.

AdPulse 열기
2

Meta Ads 연결 클릭

화면 우측 상단 또는 좌측 사이드바 하단에 있는 "Meta Ads 연결" 버튼을 클릭합니다.

Meta Ads 연결 버튼 위치우측 상단 파란색 버튼 또는 사이드바 하단 계정 영역
3

Meta(Facebook) 계정으로 로그인

팝업창에서 Facebook 계정으로 로그인합니다. 요청하는 권한 항목에 모두 허용을 눌러야 광고 데이터를 불러올 수 있습니다.

광고 계정 접근 → 허용캠페인, 광고세트, 광고소재 조회에 필요
광고 성과 지표 접근 → 허용클릭, 노출, 전환, ROAS 등 인사이트 조회에 필요
4

광고 계정 선택

로그인 후 내 광고 계정 목록이 표시됩니다. Step 4에서 만든 광고 계정을 선택합니다.

5

연결 완료 — 실시간 데이터 확인

연결이 완료되면 대시보드에 실시간 광고 데이터가 표시됩니다. 사이드바 하단에 연결된 계정 이름이 나타납니다.

연결 성공 확인사이드바 하단 초록 점 + 계정명 표시 → 정상 연결
연결 정보는 자동 저장됩니다. 브라우저를 닫았다가 다시 열어도 재로그인 없이 바로 데이터가 표시됩니다.
8
Meta 픽셀 설치
웹사이트 방문자와 구매를 추적해 광고 성과를 정확히 측정합니다
1

이벤트 관리자 접속

Meta 이벤트 관리자에서 픽셀을 생성합니다.

이벤트 관리자
2

픽셀 생성

데이터 소스 연결 → 웹 → Meta 픽셀 선택 → 픽셀 이름 입력 (예: 내 쇼핑몰 픽셀) → 픽셀 만들기

3

설치 방법 선택

사용 중인 플랫폼에 따라 설치 방법을 선택합니다.

Shopify 사용 중파트너 통합 → Shopify 선택 → 스토어 연동 (코드 없음)
WordPress 사용 중파트너 통합 → WordPress 또는 Meta Pixel 플러그인 설치
Cafe24 / 카페24마케팅 → SNS 연동 → Facebook 픽셀 ID 입력
직접 설치 (개발자)수동 설치 → 픽셀 코드를 <head> 안에 붙여넣기
4

주요 이벤트 설정

비즈니스 목표에 맞는 전환 이벤트를 추가합니다.

5

픽셀 작동 확인

Chrome에서 Meta Pixel Helper 확장 프로그램을 설치하고 내 사이트에 접속하면 픽셀이 정상 작동하는지 바로 확인됩니다.

이벤트 관리자에서 확인픽셀 상태 → 활성 (초록 점) 이면 정상 작동
주요 이벤트 목록
이벤트언제 실행주요 업종
Purchase구매 완료 페이지쇼핑몰 필수
AddToCart장바구니 담기 버튼 클릭쇼핑몰

픽셀을 설치하지 않아도 AdPulse 기본 기능은 모두 사용할 수 있습니다. 단, 전환 추적과 ROAS를 정확히 보려면 픽셀이 필요합니다.
자주 묻는 질문
설정 중 막히는 부분이 있다면 확인해 보세요
광고 계정이 비활성화됐어요
결제 수단 문제이거나 Meta 정책 위반으로 비활성화될 수 있습니다. Meta 고객센터에 검토 요청을 하거나, 결제 정보를 업데이트하세요.
광고 관리자에서 광고가 안 만들어져요
결제 수단이 등록되지 않았거나, 페이스북 페이지가 광고 계정에 연결되지 않은 경우입니다. Step 5의 결제 수단 등록과 Step 3의 페이지 연결을 다시 확인해 주세요.
AdPulse에서 데이터가 안 나와요
집행 이력이 있는 광고가 없거나, 연결한 광고 계정에 데이터가 없을 수 있습니다. 광고 관리자에서 실제 캠페인이 집행된 기간이 있는지 확인해 주세요.
인스타그램 로그인 시 "프로필을 찾을 수 없다"고 나와요
인스타그램 앱이 설치된 기기에서 로그인 링크를 열면 앱이 가로챕니다. Safari나 Chrome에서 직접 접속하거나, 인스타그램 앱이 없는 PC에서 시도하세요.
카드 결제가 계속 실패해요
카드사에서 해외 온라인 결제를 막아두었을 가능성이 큽니다. 카드 앱이나 고객센터에서 해외 인터넷 결제 허용으로 변경 후 다시 시도하세요. VISA/Mastercard 브랜드 카드를 사용하는 것이 안전합니다.
픽셀 Helper에서 픽셀이 감지되지 않아요
픽셀 코드가 <head> 태그 안에 정확히 삽입되었는지 확인하세요. GTM을 사용하는 경우 GTM 컨테이너가 게시(Publish)되었는지 확인하세요. 브라우저 캐시를 지우고 다시 접속해도 나타나지 않으면 픽셀 ID가 올바른지 이벤트 관리자에서 다시 확인하세요.
여러 광고 계정을 관리하고 싶어요
AdPulse에서 재연결 버튼을 눌러 다른 광고 계정을 선택하면 전환됩니다. 한 번에 하나의 계정 데이터를 조회합니다.
