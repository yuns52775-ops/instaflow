1
Meta 비즈니스 계정 설정
광고 집행과 API 연동의 기반이 되는 비즈니스 계정을 준비합니다
1

Meta Business Suite 접속

아래 주소로 이동하여 Facebook 계정으로 로그인합니다.

business.facebook.com
2

비즈니스 포트폴리오 생성

처음 접속 시 새 비즈니스 포트폴리오 만들기를 클릭합니다. 이미 있다면 이 단계를 건너뜁니다.

비즈니스 이름: 회사명 또는 브랜드명 비즈니스 이메일: 회사 이메일 주소 비즈니스 웹사이트: 공식 웹사이트 URL
3

비즈니스 인증 (권장)

설정 → 비즈니스 정보 → 비즈니스 인증을 완료하면 API 제한이 크게 완화됩니다. 사업자등록증 또는 기업 서류가 필요합니다.

기존 계정 활용: 이미 Meta Business Manager를 사용 중이라면 Step 2로 바로 이동하세요. 개인 Facebook 광고 계정만 있는 경우에도 API 연동이 가능합니다.
2
광고 계정 생성 및 ID 확인
AdPulse 연결에 필요한 광고 계정 ID를 확인합니다
1

광고 계정 생성 또는 확인

Business Suite → 광고 계정 → 새 광고 계정 추가 또는 기존 계정 선택

business.facebook.com/settings/ad-accounts
2

광고 계정 ID 확인

광고 계정을 클릭하면 상단에 광고 계정 ID가 표시됩니다. act_ 접두어를 포함한 숫자 ID입니다.

예시: act_1234567890123456 ↑ 이 전체 형식 또는 숫자만 메모해두세요
3

광고 관리자에서도 확인 가능

광고 관리자(Ads Manager) 접속 후 URL에서도 확인할 수 있습니다.

URL 예시: https://www.facebook.com/adsmanager/manage/campaigns?act=1234567890123456 ↑ 이 숫자가 광고 계정 ID
광고 계정 ID 형식
act_XXXXXXXXXXXXXXXX
API 호출 시 이 형식 사용
숫자만 입력해도 OK
XXXXXXXXXXXXXXXX
AdPulse가 자동으로 act_ 추가
결제 수단 필수: 광고 계정에 결제 수단이 등록되어 있어야 API를 통해 데이터를 불러올 수 있습니다. 미등록 시 일부 데이터 조회가 제한됩니다.
3
Meta Developer App 설정
API 액세스를 위한 앱을 생성하고 필요한 권한을 설정합니다
1

Meta for Developers 접속

developers.facebook.com/apps
2

새 앱 만들기

앱 만들기 클릭 → 앱 유형을 "비즈니스"로 선택합니다.

앱 이름: AdPulse (또는 원하는 이름) 앱 연락처 이메일: 본인 이메일 비즈니스 포트폴리오: Step 1에서 만든 포트폴리오 선택
3

마케팅 API 추가

앱 대시보드 → 제품 추가 → 마케팅 API → "설정" 클릭

4

필요 권한(Scopes) 설정

앱 설정 → 고급 설정 → 권한 또는 Graph API Explorer에서 아래 권한을 추가합니다.

권한명 용도 필요 여부
ads_read 광고 캠페인, 광고세트, 광고소재 조회 필수
ads_management 광고 데이터 전체 액세스 (인사이트 포함) 필수
read_insights 광고 성과 지표 및 인사이트 조회 필수
business_management 비즈니스 자산 및 광고 계정 목록 조회 필수
instagram_basic 인스타그램 계정 기본 정보 선택
pages_read_engagement 페이스북 페이지 인사이트 선택
앱 라이브 전환
개인정보처리방침 URL 등록 앱 설정 → 기본 설정 → 개인정보처리방침 URL 입력 (AdPulse 사이트의 /privacy 페이지)
앱 아이콘 업로드 1024×1024px PNG 이미지 업로드 필요
라이브 전환 앱 대시보드 상단 토글 → 개발 모드 → 라이브로 전환. 자신의 계정에서만 사용 시 개발 모드도 가능.
개발 모드로도 충분: 본인 계정의 광고 데이터만 조회할 경우 앱 검수 없이 개발 모드에서도 모든 기능이 작동합니다. 다른 사람의 계정에 접근할 때만 라이브 전환이 필요합니다.

액세스 토큰 발급
API 인증에 사용할 장기 액세스 토큰을 발급합니다
1

Graph API Explorer 접속

developers.facebook.com/tools/explorer
2

앱 선택 및 권한 추가

우측 상단 드롭다운에서 Step 3에서 만든 앱을 선택합니다.
권한 추가 버튼 클릭 → 아래 권한들을 체크합니다.

☑ ads_read ☑ ads_management ☑ read_insights ☑ business_management
3

토큰 생성

새 액세스 토큰 생성 클릭 → Facebook 로그인 팝업 → 모든 권한 허용 → 토큰이 생성됩니다.

4

단기 토큰 → 장기 토큰 변환 (중요!)

Graph API Explorer에서 발급된 토큰은 1~2시간만 유효합니다. 아래 API를 호출하여 60일짜리 장기 토큰으로 변환하세요.

GET https://graph.facebook.com/v21.0/oauth/access_token ?grant_type=fb_exchange_token &client_id={앱_ID} &client_secret={앱_시크릿} &fb_exchange_token={단기_토큰} → 응답의 access_token 값이 장기 토큰입니다
5

앱 ID / 앱 시크릿 확인 위치

Meta for Developers → 해당 앱 → 앱 설정 → 기본 설정

앱 ID: 앱 설정 > 기본 설정 상단에 표시 앱 시크릿 코드: 앱 설정 > 기본 설정 > "표시" 클릭
토큰 만료 자동 감지: AdPulse는 토큰이 만료되면 자동으로 감지하여 재연결 화면을 표시합니다. 60일마다 토큰을 재발급해야 합니다.
보안 주의: 액세스 토큰은 광고 계정 접근 권한을 가집니다. 타인에게 공유하거나 공개된 곳에 게시하지 마세요. AdPulse는 토큰을 브라우저 localStorage에만 저장하며 서버로 전송하지 않습니다.

Meta 픽셀 설정
전환 추적을 위한 픽셀을 생성하고 웹사이트에 설치합니다
1

이벤트 관리자 접속

business.facebook.com/events_manager
2

픽셀 생성

데이터 소스 연결 → 웹 → Meta 픽셀 → 픽셀 이름 입력 → 생성

픽셀 이름 예시: [브랜드명] Website Pixel 웹사이트 URL: https://yoursite.com
3

픽셀 코드 설치 방법 선택

설치 방식을 선택합니다. GTM 사용 중이라면 Google Tag Manager를 권장합니다.

옵션 A: 직접 설치 (수동으로 <head> 태그에 삽입) 옵션 B: 파트너 통합 (Shopify, WordPress, GTM 등) 옵션 C: 이메일로 개발자에게 전달
4

픽셀 기본 코드 설치 (직접 설치 시)

모든 페이지의 <head> 태그 안에 아래 코드를 삽입합니다.

<!-- Meta Pixel Code --> <script> !function(f,b,e,v,n,t,s){if(f.fbq)return;n=f.fbq=function(){...}; ... }(window, document,'script','https://connect.facebook.net/en_US/fbevents.js'); fbq('init', '{픽셀_ID}'); ← 여기에 본인 픽셀 ID 입력 fbq('track', 'PageView'); </script> <!-- End Meta Pixel Code -->
5

전환 이벤트 설정

비즈니스 목표에 맞는 이벤트를 추가로 설치합니다.

// 구매 완료 fbq('track', 'Purchase', {value: 29900, currency: 'KRW'}); // 장바구니 담기 fbq('track', 'AddToCart', {value: 29900, currency: 'KRW'}); // 결제 시작 fbq('track', 'InitiateCheckout'); // 회원가입 fbq('track', 'CompleteRegistration'); // 리드 (문의/신청) fbq('track', 'Lead'); // 콘텐츠 조회 fbq('track', 'ViewContent', {content_name: '상품명'});
6

Meta 픽셀 헬퍼 활용: Chrome 확장 프로그램 "Meta Pixel Helper"를 설치하면 픽셀이 올바르게 동작하는지 실시간으로 확인할 수 있습니다.

❓ 자주 묻는 질문
Q. 데이터가 전혀 안 나와요
광고 계정에 집행 내역이 있는지 확인하세요. 신규 계정이나 일시정지 계정은 일부 데이터가 없을 수 있습니다.
Q. 픽셀 페이지에 이벤트가 안 보여요
픽셀이 광고 계정에 연결되어 있어야 합니다. 이벤트 관리자 → 픽셀 → 설정 → 광고 계정 연결을 확인하세요.
Q. 경쟁사 광고 검색이 안 돼요
Meta Ad Library API는 별도의 접근 신청이 필요할 수 있습니다. Meta Ad Library 페이지에서 직접 검색하는 것도 가능합니다.
Q. 토큰을 60일마다 갱신해야 하나요?
네. 장기 토큰은 60일 유효합니다. AdPulse가 만료 시 자동으로 알려주며, 같은 방법으로 재발급하면 됩니다.
Q. 여러 광고 계정을 동시에 볼 수 있나요?
현재 버전은 계정 1개씩 조회합니다. 다른 계정을 보려면 재연결 버튼으로 계정을 전환하세요.
